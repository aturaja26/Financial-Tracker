# Financial Tracker Premium 💰

Selamat datang di **Financial Tracker Premium**, solusi mutakhir untuk mencatat, merencanakan, dan melacak kesehatan finansial pribadi Anda dengan antarmuka yang sangat modern dan cepat!

Aplikasi web ini dirancang secara eksklusif menggunakan **HTML, CSS, dan JavaScript murni** (100% Statis), yang membuatnya sangat ringan, responsif, dan dapat langsung dijalankan dari browser atau di-hosting di *GitHub Pages* tanpa memerlukan *backend server* yang rumit.

---

## ✨ Keunggulan Utama Aplikasi

Kenapa Anda harus menggunakan Financial Tracker Premium?

1. **Sangat Cepat & Ringan (Zero Backend)**
   Tanpa waktu tunggu pemuatan (loading). Seluruh data Anda disimpan secara instan di perangkat Anda sendiri (*LocalStorage*), menjamin privasi maksimal dan akses yang kilat.
   
2. **Desain Premium Ala Fintech**
   Tinggalkan spreadsheet yang membosankan! Aplikasi ini menawarkan antarmuka UI/UX yang memanjakan mata, mendukung **Dark Mode** & **Light Mode**, serta menyertakan interaksi *micro-animation* yang mulus.

3. **Dashboard Pintar & Komprehensif**
   Dalam satu layar, Anda bisa melihat ringkasan kekayaan bersih (*Net Worth*), total pendapatan, pengeluaran, serta visualisasi grafik indah (Pie, Bar, Line Chart) berbasis *Chart.js*.

4. **Financial Health Score (Skor Kesehatan Finansial)**
   Secara otomatis menghitung skor finansial Anda (0-100) berdasarkan kebiasaan menabung (*Saving Rate*), rasio utang, rasio arus kas, dan ketersediaan dana darurat.

5. **Responsif Sepenuhnya (Mobile Friendly)**
   Dirancang dari awal untuk bisa diakses dengan nyaman melalui layar *Smartphone*, *Tablet*, hingga *Desktop*. Anda bisa mencatat pengeluaran dari genggaman tangan di mana saja.

---

## 🚀 Fitur - Fitur Andalan

- **💳 Manajemen Transaksi**: Catat pemasukan (Gaji, Freelance, Bonus) dan pengeluaran (Makanan, Transportasi, Hiburan) lengkap dengan kategori.
- **🎯 Budget Planner**: Tentukan batas pengeluaran per kategori. Sistem akan memberi peringatan visual (hijau, kuning, merah) jika Anda hampir melewati budget.
- **🐷 Savings Goals**: Buat target tabungan (seperti: "Liburan ke Jepang" atau "Beli Laptop") dan pantau *progress bar* pencapaiannya!
- **📈 Investment Tracker**: Lacak aset investasi Anda (Saham, Reksa Dana, Kripto) beserta keuntungan/kerugian (Return/Loss) secara real-time.
- **📉 Debt Tracker**: Punya cicilan atau utang? Catat total utang, sisa pokok, dan cicilan bulanan agar pelunasan tetap terpantau.
- **🔁 Subscriptions**: Kelola tagihan langganan bulanan Anda (Netflix, Spotify, Gym) untuk mencegah "kebocoran halus" pada dompet Anda.

## 🛠 Cara Penggunaan (Bisa Langsung Coba!)

Karena aplikasi ini sepenuhnya statis, Anda tidak perlu menginstal server apapun!

1. Buka folder proyek ini.
2. Klik dua kali pada file **`register.html`** untuk membukanya di browser (Chrome/Edge/Safari/Firefox disarankan).
3. Buat akun pertama Anda (data aman di *LocalStorage* perangkat Anda).
4. **Login**, dan selamat menikmati seluruh fitur pencatatan! 

*Catatan: Jika Anda ingin membagikannya, Anda cukup mengunggah repositori ini ke **GitHub** dan mengaktifkan fitur **GitHub Pages**.*
